# Virtualization & Cloud Infrastructure Lab

This repository documents a collection of hands on infrastructure, virtualization, networking, and cloud operations projects focused on building practical experience with Linux systems, virtualization platforms, containerization, Infrastructure as Code concepts, and operational troubleshooting workflows.

The lab environment was built using QEMU virtualization, Alpine Linux, Docker, Kubernetes related tooling, Git, and MSYS2/MinGW64 on Windows hosts. The projects within this repository explore core infrastructure concepts including virtual machine provisioning, networking, storage management, snapshots and cloning, SSH access, observability, reliability analysis, and infrastructure troubleshooting.
