| VDI Component           | On-Prem Responsibility                                                   | DaaS Equivalent (Azure Virtual Desktop / Amazon WorkSpaces)           | Service Model | Provider Responsibility                         | Customer Responsibility                                  |
| ----------------------- | ------------------------------------------------------------------------ | --------------------------------------------------------------------- | ------------- | ----------------------------------------------- | -------------------------------------------------------- |
| Connection Broker       | IT installs and maintains broker servers and manages session routing     | Managed control plane that handles authentication and session routing | SaaS          | Runs and maintains broker service               | Configure access policies and user assignments           |
| Desktop Pools           | IT creates and maintains pools of persistent and non-persistent desktops | Cloud-hosted virtual desktop pools                                    | IaaS          | Provides VM infrastructure and availability     | Choose VM size, configure pools, manage scaling settings |
| Session Hosts           | IT installs OS, patches systems, and manages virtualization hosts        | Cloud virtual machines running desktop sessions                       | IaaS          | Provides compute infrastructure                 | Manage OS configuration, updates, and installed software |
| Remote Display Protocol | IT configures and supports RDP or other display protocol                 | Built-in managed protocol service                                     | SaaS          | Operates and optimizes display protocol service | Configure connection settings and user access            |
| User Profile Management | IT manages roaming profiles or file servers                              | Managed storage services for profiles                                 | PaaS          | Operates storage platform                       | Manage user data and profile settings                    |
| Image Management        | IT builds and maintains master desktop images                            | Cloud image gallery and deployment system                             | IaaS          | Provides storage and image distribution tools   | Create, update, and secure the base image                |
